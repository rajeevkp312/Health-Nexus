## üöÄ Getting Started

### Prerequisites
- Node.js 16+ and npm/yarn
- Backend API server running
- Modern web browser

### Installation
1. Navigate to the admin directory
2. Install dependencies: `npm install`
3. Start development server: `npm run dev`
4. Access admin at: `http://localhost:3000/admin`
